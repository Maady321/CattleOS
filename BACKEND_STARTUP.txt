# CattleOS Backend Startup Guide

Follow these steps to get the FastAPI backend up and running.

## Prerequisites
- Python 3.10+
- PostgreSQL (or Supabase)
- Redis (optional, for Celery tasks)

## Local Development Setup

### 1. Create and Activate Virtual Environment
If you haven't already:
```bash
cd backend
python -m venv venv
```

Activate the environment:
- **Windows**: `venv\Scripts\activate`
- **Mac/Linux**: `source venv/bin/activate`

### 2. Install Dependencies
```bash
pip install -r requirements.txt
```

### 3. Environment Variables
Ensure you have a `.env` file in the `backend` directory. Use `.env.example` as a template if available.
Key variables:
- `DATABASE_URL`: Your PostgreSQL connection string
- `SECRET_KEY`: For JWT authentication

### 4. Database Migrations
Run the following to set up your database schema:
```bash
alembic upgrade head
```

### 5. Start the Server
Run the FastAPI application using Uvicorn:
```bash
uvicorn app.main:app --reload
```
The API will be available at: `http://localhost:8000`
Swagger Documentation: `http://localhost:8000/docs`

---

## Running with Docker
If you prefer Docker, ensure Docker Desktop is running and execute:
```bash
docker-compose up --build
```
This will start the backend, database, and any required services.
